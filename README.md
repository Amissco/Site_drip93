# Site_drip93

Site web catalogue de vêtements permettant de parcourir les articles disponibles et de composer un panier. Pour passer commande, il suffit de contacter le vendeur directement sur Snapchat.

## Fonctionnalités
- Catalogue de vêtements avec filtrage par catégorie
- Panier interactif
- Commande via contact Snapchat : **elcherif93vg**

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (avec claude)
